About

An ecosystem built from scratch on Arc

StableLab is an independent experiment: real smart contracts written and deployed on Arc Network Testnet. It started as a simple "stake, borrow, swap" idea; over time it grew into an eight-module ecosystem that rebuilds established institutional finance concepts (invoice factoring, escrow, treasury management, continuous payments) on Arc.
