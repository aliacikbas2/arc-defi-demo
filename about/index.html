About

An ecosystem of code, built by someone who doesn't code

StableLab is an independent experiment built by someone with no software background, writing and deploying real smart contracts on Arc Network Testnet. It started as a simple "stake, borrow, swap" idea; over time it grew into an eight-module ecosystem that rebuilds established institutional finance concepts (invoice factoring, escrow, treasury management, continuous payments) on Arc.
